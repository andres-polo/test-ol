# Frontend - Comercio (Angular 18)

## Requisitos

- Node.js 18+
- API backend en ejecución (`https://localhost:5242` por defecto)

## Desarrollo

```bash
npm install
npm start
```

Abrir `http://localhost:4200`

## Build producción

```bash
npm run build
```

Salida: `dist/frontend/`. Comprimir para entrega según la prueba.

## Tests

```bash
npm test
```

## Credenciales de prueba

- Admin: `admin@comercio.com` / `Admin123!`
- Auxiliar: `auxiliar@comercio.com` / `Auxiliar123!`

## Configuración API

Editar `src/environments/environment.ts` para cambiar la URL de la API.
